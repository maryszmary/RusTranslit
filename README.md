# RusTranslit

## Installation

### Build module

```sh
git clone https://github.com/maryszmary/RusTranslit
cd RusTranslit
pip install -e .
```

## Usage

Example code:

```python
from rus_translit import transliterate

transliterate('is this a real life? is this just fantasy?')
```

Example output:
```python
'из зис а рил лайф? из зис джаст фентеси?'
```
